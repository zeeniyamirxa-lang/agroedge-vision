# 🌿 AgroEdge-Vision

An AI-powered plant disease diagnostic app built with **Streamlit** and **PyTorch**.

Upload or photograph a leaf and get an instant diagnosis powered by a fine-tuned ResNet-50 model trained on 38 plant disease classes.

## Features
- 📸 Live camera scan or image upload
- 🧠 ResNet-50 deep learning model
- 📊 Scan history & analytics dashboard
- ☁️ Cloud database via Supabase
